# はとぴ図鑑

Heartopia（ハートピアスローライフ）の魚・虫・野鳥の出現情報を
リアルタイムで確認できる非公式ファンツールです。

## 主な機能

- 現在の天気・時間帯に出現する生き物をリアルタイム表示
- 魚・虫・野鳥の種類別フィルター
- 出現モード切替（全表示 / 今の天気限定 / 今は出ないもの）
- 図鑑順・レベル順・未コンプ順・未認証順での並び替え
- レベル範囲の絞り込み
- 名前・出現場所での検索
- ⭐ コンプリートチェック機能（一括チェック対応）
- 🎖 認証マスター記録機能（一括チェック対応）
- コンプ状況のシェア画像生成・X（Twitter）投稿
- ダークモード対応
- PWA対応（ホーム画面に追加して使用可能）

## 使い方

ブラウザでアクセスするだけで使えます。
ホーム画面に追加するとアプリのように起動できます。

🔗 https://ram1x7.github.io/heartopilist/

## 注意事項

- 本ツールは個人が制作した非公式のファンツールです
- ゲーム公式・開発元とは一切関係ありません
- データはブラウザ内（LocalStorage）のみで管理されます

# Hatopi Zukan (はとぴ図鑑)

An unofficial fan tool for Heartopia (Heartopia Slow Life) that lets you
check which fish, bugs, and birds are appearing in real time.

## Features

- Real-time display of creatures based on current weather and time of day
- Filter by type: fish, bugs, or birds
- Appearance mode switching (All / Current weather only / Not appearing now)
- Sort by: Pokédex order / Level / Incomplete first / Uncertified first
- Level range filter
- Search by name or location
- ⭐ Completion check (bulk check supported)
- 🎖 Authentication Master record (bulk check supported)
- Share card image generation & X (Twitter) post
- Dark mode support
- PWA support (add to home screen)

## How to Use

Just open the link in your browser — no installation needed.
You can also add it to your home screen to use it like an app.

🔗 https://ram1x7.github.io/heartopilist/

## Disclaimer

- This is an unofficial fan-made tool and is not affiliated with
  the official game or its developers in any way.
- All data is stored locally in your browser (LocalStorage) only.
